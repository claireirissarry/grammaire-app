# Grammaire Mondial 2026 - Objectif Finale

Une application éducative interactive sur la grammaire française avec un thème football/Mondial 2026.

## 🎯 Fonctionnalités

- 5 modules d'apprentissage de grammaire
- 3 niveaux de difficulté (Découverte, Entraînement, Défi)
- Questions interactives (QCM, remplissage, réponses ouvertes)
- Système de badges et de points XP
- Tableau de bord de progression
- Espace parents pour suivre les progrès

## 🚀 Déploiement

Cette application est déployée sur Vercel. Visitez le lien pour y accéder.

## 📝 Modules

1. L'article défini / article indéfini
2. L'accord de l'adjectif qualificatif
3. Leur vs Leurs
4. Forme négative & forme affirmative
5. Transposition au futur simple
